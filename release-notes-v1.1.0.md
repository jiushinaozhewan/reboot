## Reboot v1.1.0

本次版本聚焦在三个方向：稳定性修复、多目标易用性、双端取证能力。

### 亮点

- `reboot-agent.exe`
  - 修复托盘菜单在端口设置、显示密钥、修改密钥之间切换后出现假死的问题
  - 输入对话框改为独立 helper 进程，去掉黑框并降低瞬时资源占用
  - 新增“调试日志”开关，默认关闭，调试时写入程序目录 `log.txt`

- `reboot-client.exe`
  - 支持每目标独立密钥并本地加密保存
  - 修复目标密钥输入框串行联动与异常显示问题
  - 新增自动扫描：支持指定网段 + 指定端口，自动填充目标 IP，并按需新增条目
  - 新增“调试日志”开关，默认关闭，调试时写入程序目录 `log.txt`

### 适合更新的场景

- 之前遇到 Agent 托盘菜单无响应、假死
- 需要批量维护多台设备
- 需要更清晰地排查“连接成功但命令失败”之类的问题
- 希望平时零日志、出问题时再打开取证
